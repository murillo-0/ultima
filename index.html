Circulo Cromático















O que é circulo cromático?

O círculo cromático é uma ferramenta que representa de forma simplificada as cores a olho nu, e ajuda a entender sobre as possíveis combinações de diferentes cores e suas propriedades. Na prática, ela ajuda a desenvolver uma paleta de cor para as criações.

Como usar o círculo cromático?

O círculo cromático é representado por um gráfico redondo dividido em 12 partes iguais, sendo que cada uma dessas partes representa uma cor (com diversos níveis de saturação). São três cores primárias, três cores secundárias e seis cores terciárias.


Inicialmente, é possível fazer quatro tipos de combinações de cores diferentes:

  • Complementares:

    São cores que ficam opostas uma às outras no círculo, como verde e vermelho ou azul e laranja;

  • Cores análogas:

    São cores com alto grau de proximidade, bem próximas uma das outras. Utiliza uma cor primária e outras duas “vizinhas” a ela no círculo.

  • Tríade de cores:

    São três cores que têm a mesma distância no círculo, com um triângulo equilátero, onde todos os lados são iguais;

  • Monocromatismo:

    É o famoso tom sobre tom, utilizando vários tons diferentes da mesma cor no círculo.
